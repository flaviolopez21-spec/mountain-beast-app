How Apple Watch sync works with GitHub

GitHub Pages hosts the website, but it cannot access HealthKit. On iPhone, export Health data from the Health app, unzip the export, and choose apple_health_export/export.xml above.

True automatic sync requires a signed native iPhone app built with Xcode, HealthKit permissions, and an App Store or personal-device installation. The native app can then pass selected records into this local tracker. Publishing these files to GitHub does not enable that permission.

Because this version has no cloud account, iPhone and computer data remain separate. Use private backup export and restore to move data between them. Do not commit personal backup or Health XML files to a public repository.
